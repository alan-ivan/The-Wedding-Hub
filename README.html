The Wedding Hub

About

The Wedding Hub is a comprehensive tool designed to simplify wedding planning. It offers features like guest management, task tracking, budget allocation, vendor coordination, seating charts, and license management.

Features

  • Guest Management
  • Task Tracking
  • Budget Allocation
  • Vendor Coordination
  • Seating Chart Management
  • License Activation and Management

Hosted Pages

  • Privacy Policy
  • Terms and Conditions

Getting Started

Clone this repository and open the HTML files in your browser to view the documents.

        
          git clone https://github.com/your-username/weddinghub-docs.git

To host the pages, upload them to a web server or use GitHub Pages by enabling it in your repository settings.
